# 📚 My Library

A simple library web app built with **HTML, CSS, and JavaScript**.  
This project lets you add, display, remove, and update the read status of books.  

---

## 🚀 Features
- Add new books with **Title, Author, Pages, and Read Status**.
- View books in **Table View** or **Card View** (toggle between them).
- Remove books from the library.
- Toggle a book’s **read status** with one click.
- Modal dialog form for adding books using the native **`<dialog>` element**.
- **Prototype method** (`Book.prototype.toggleRead`) used to toggle read status.
- **Dataset attributes (`data-id`)** used to uniquely identify and manage books.
- Responsive design with clean UI.

---

## 🛠️ Tech Stack
- **HTML5** for structure  
- **CSS3** for styling  
- **JavaScript (ES6)** for functionality
